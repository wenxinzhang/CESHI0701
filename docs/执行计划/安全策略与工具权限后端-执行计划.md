## 3. 全局技术约定（对齐现有代码）

| 项 | 约定 |
|----|------|
| 后端根目录 | `server/` |
| ORM | Prisma 6，`schema.prisma` 单文件 |
| 表命名 | 模型 PascalCase + `@@map("snake_case")`；列驼峰不加 `@map` |
| 主键 | `id Int @id @default(autoincrement())` |
| 公共字段 | `tenantId Int?` + `createTime DateTime @default(now())` + `updateTime DateTime @updatedAt`（**无软删除、无 createBy**） |
| 枚举 | `String @default(...) @db.VarChar(n)` + catalog 白名单 + DTO `@IsIn` 校验 |
| 数组/对象 | Prisma `Json` 类型 |
| 索引 | `@@index([tenantId])` + 高频筛选字段 + 时间字段 |
| Controller | `@Controller('admin/xxx')` + 全 `@Post` + `@Perms('action')`，继承 `BaseController`，`this.ok(data)` 返回 |
| Service | 继承 `BaseService`，构造注入 `PrismaService` 并 `super(prisma, '模型委托名')` |
| DTO | class-validator + `@ApiProperty`，每查询 DTO 自带 `page?/pageSize?` |
| 分页出参 | `{ list, pagination: { page, pageSize, total } }` |
| 当前用户 | `@Admin('userId')` / `@Admin('username')` |
| 模块注册 | 新建 `src/modules/<name>/<name>.module.ts` 自动装配，**不改 app.module** |
| schema 变更 | `npm run prisma:push`（同步 `///` 注释到列 comment） |
| 后端构建验证 | `cd server && npx tsc --noEmit`（或项目 build 命令） |
| 前端构建验证 | `cd frontend && npx vite build` |
